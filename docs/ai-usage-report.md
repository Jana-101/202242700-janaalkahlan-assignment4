# AI Usage Report – Assignment 4

## Tools Used & How They Were Used

### ChatGPT

ChatGPT was used throughout the project, especially during the **planning, design, and refinement stages**.

* I began by defining a general structure and basic features for my portfolio.
* I used ChatGPT to:

  * Help set up the GitHub repository and organize the project structure
  * Expand and improve my initial feature ideas into more advanced and relevant functionality
  * Suggest enhancements required for Assignment 4 (usability improvements, clearer user guidance, stronger UX)
  * Identify weaknesses from previous assignments and propose improvements
  * Provide guidance on improving user interaction and navigation clarity
  * Assist in writing and refining documentation (README, technical documentation, AI report)

* ChatGPT was also used to review the overall system and suggest improvements in:
  * usability
  * feature clarity
  * user guidance
  * consistency across the application

---

### Claude

Claude was primarily used during the **implementation and coding phase**.

* I provided my existing code from Assignment 3, which was functional but not fully optimized.
* Claude helped to:

  * Clean and restructure the codebase
  * Improve readability and organization of JavaScript logic
  * Refine CSS styling and visual layout
  * Implement additional features required for Assignment 4
  * Improve existing features (filtering, search, sorting, API handling)
  * Fix bugs and resolve logic issues
  * Enhance interactivity and user experience
  * Improve validation behavior and feedback clarity

---

## How AI Was Used in the Workflow

1. I first planned the application structure and features independently  
2. I used ChatGPT to refine and expand those ideas into more advanced features  
3. I implemented the base version of the application  
4. I used Claude to clean, restructure, and enhance the code  
5. I iteratively tested, reviewed, and modified all AI outputs  
6. I used ChatGPT again to refine usability, documentation, and final improvements  

AI was used as a **support tool throughout development**, not as a direct solution generator.
