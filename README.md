# Personal Web Application – Assignment 4

## 📌 Project Description

This project is the final and most advanced version of the personal portfolio developed across previous assignments.

It transforms the portfolio into a more complete **personal web application**, focusing on strong interactivity, dynamic data handling, improved usability, and a more professional user experience.

The goal of this assignment is to demonstrate advanced frontend development skills by building a polished, feature-rich, and user-centered web application.

---

## 🛠 Technologies Used

* HTML  
* CSS  
* JavaScript  
* GitHub REST API  
* localStorage  

---

## ✨ Features

### Core Features

* Fully responsive design (desktop, tablet, mobile)
* Navigation with hamburger menu
* Light/Dark theme toggle with persistence
* Project filtering by category
* Live search functionality
* Contact form with validation and feedback

---

### Assignment 3 Features (Improved)

* GitHub API integration (live repositories display)
* Project sorting (A–Z, Z–A, category)
* Persistent state (theme, filter, search saved)
* User personalization (name input + welcome banner)
* Session timer tracking time spent
* “How to Use” guide for better usability
* Improved form validation and UX feedback

---

### Assignment 4 Enhancements

* Improved overall user experience and clarity across all sections  
* Cleaner and more professional UI design and layout  
* Enhanced guidance for users on how to navigate and interact with the website  
* Fixed usability issues (form behavior, feedback clarity, UI consistency)  
* Better structured and optimized JavaScript logic  
* Improved code organization and maintainability  
* Refined interaction flows for a smoother experience  

---

## 🧭 How to Use the Website (User Guidance)

1. Use the navigation menu to move between sections:
   **About, Projects, Skills, Contact**

2. In the Projects section:
   - Use the **search bar** to find projects  
   - Use **filters** to view specific categories  
   - Use **sorting** to organize results  

3. Scroll to the GitHub section:
   - View live repositories fetched from GitHub  

4. Use the theme toggle:
   - Switch between light and dark modes  

5. Interact with personalization features:
   - Enter your name or skip  
   - Receive a welcome message  

6. Fill out the contact form:
   - Enter valid inputs  
   - Submit to see feedback
